My Group's Game Proposal

Grace Adeboyejo / August 2,2021

On Friday, July 30,2021, My group created a proposal for a game that would help teachers create more inclusive spaces for gender identity.

1. Our game was based of the game 'Amoung Us'. In Amoung Us, you have task you have to complete as crewmemeber. So, we decided that the teachers would have task to remeber the pronounsof all of their students.

2. In Amoung Us there are different maps you can play on and our map was a classroom.

The avatars in our game were ranomly assigned so that teachers could get used to seeing different combinations of "students". This means that the pronouns did not match up with the avatar so the teacher could remember the student's pronouns from memory and not based on what the avatar looked like.
